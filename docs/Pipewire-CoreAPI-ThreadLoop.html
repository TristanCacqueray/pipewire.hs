Pipewire.CoreAPI.ThreadLoop

Description

The threaded loop enable using the binding from multiple thread.

Actions needs to be wrapped using withLock, and to let the loop run, for example to take a MVar, then use withUnlock.

TODO: investigate managing the thread loop manually, e.g. by waiting for `IO ()` action and pausing the loop with pw_main_loop_quit
